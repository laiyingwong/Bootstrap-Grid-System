Responsive Images

Images in Bootstrap are made responsive with .img-fluid. This applies max-width: 100%; and height: auto; to the image so that it scales with the parent width.

The three images below start out stacked. The first two become horizontal and each takes up 50% of the row at the medium breakpoint with col-md-6.

When the viewport hits the large breakpoint with col-lg-4, all three images will take up the row, and each occupies 1/3 of the space.

We can eliminate the space between images by adding no-gutters to the row class.
